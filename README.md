# My Todo List App

Simple tool to organize everything. You can add a list, complete the tasks and delete them.

![alt tag](https://i.imgur.com/AvfJb3t.gif)

## How It's Made:

**Tech used:** HTML, CSS, JavaScript

Using a constructor to create each ToDo, and a class to handle with UI methods. Also, created function to handle with WEB API interactions.
